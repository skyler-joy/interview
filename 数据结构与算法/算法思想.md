1. ##### 反转数组

循环次数：  n-1/2   n=数组最大下标

```go
var nums []int
	//反转数组
	nums = []int{1, 2, 3, 4, 5}
	n := len(nums) - 1
	for i := 0; i <= (n-1)/2; i++ {
		nums[i], nums[n-i] = nums[n-i], nums[i]
	}

	fmt.Println(nums)
```
